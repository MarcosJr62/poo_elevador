Sistema de Elevador

Projeto desenvolvido para a disciplina de Programação Orientada a Objetos (POO), visando simular o funcionamento de um elevador.

Funcionalidades
O sistema contém as seguintes ações:

Subir andar;
Descer andar;
Abrir a porta;
Fechar a porta;
Mostrar o andar atual;
Verificar se a porta está ou não aberta.
